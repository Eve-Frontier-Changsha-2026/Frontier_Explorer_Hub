# Frontier Explorer Hub 路演 Demo（3 分钟，电影叙事版）

## 1) 开场（20 秒）
今天我们带来的是 **Frontier Explorer Hub**，一个为 EVE Frontier 打造的链上情报与指挥平台。  
它把探索、战斗、物流产生的关键信息，沉淀成可交易、可追溯、可权限控制的链上情报对象。

## 2) 痛点（25 秒）
- 玩家与联盟的情报分散在聊天和第三方表格，决策延迟高。
- 高价值航线与矿点缺乏统一威胁评估，损失不可控。
- 战役结束后复盘困难，经验无法沉淀成组织资产。

## 3) 方案（45 秒）
- 每条情报作为 Sui 对象上链，带有时间戳、位置、类型、可见范围。
- 平台通过 World Contracts 管理权限与交易，支持公共、部落、联盟分级情报。
- 外部工具通过 dApp-kit 接入实时数据，形成“发现-验证-交易-复盘”的完整循环。

## 4) 现场演示脚本（70 秒）
1. 打开首页 Hero：全屏官方 Key Art + 大字标题，抛出“情报即武器”的核心主张。  
2. 滚动到 Tactical Frames：逐张讲解官方宣传图如何映射到可执行决策。  
3. 切到 Live Ops Console：按 90 秒流程展示产品如何支持实时战术指挥。  
4. 滚动到 Finale：用发光 CTA 收束，强调平台化能力和生态可扩展性。  
5. 总结一句：我们不是做信息看板，而是在做 EVE Frontier 的决策中枢。  

## 5) 商业与生态价值（20 秒）
- 对玩家：降低死亡成本，提升决策质量。  
- 对联盟：沉淀组织知识，形成可持续战术优势。  
- 对生态：开放 dApp Store 能力，支持更多第三方工具接入。

## 6) 收尾（10 秒）
Frontier Explorer Hub 把官方叙事中的“生存与后果”，转译成可编程的实时决策系统。  
这就是我们对 EVE Frontier 外部工具赛道的最强回答。
